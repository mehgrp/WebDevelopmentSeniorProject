Practice

The exercises below are optional, and are meant to help you practice your newly learned skills. Each option increases in difficulty. There is also a video tutorial available if you would like to see an example of how to complete each practice. There is a link at the bottom of every page to additional resources that provide more help and information.

Practice 1

Declare some string variables. Print them out to the console. Declare some number variables. Print out a solution after using the variables in some math operation.

Practice 2

Declare a function with a parameter that prints out a phrase with interpolation. Call the function with at least 3 different arguments.

Practice 3

Make an array of objects. Iterate through each element to create some sort of output.(interpolation, conditionals, etc.)

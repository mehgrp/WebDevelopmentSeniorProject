Practice

The exercises below are optional, and are meant to help you practice your newly learned skills. Each option increases in difficulty. There is also a video tutorial available if you would like to see an example of how to complete each practice. There is a link at the bottom of every page to additional resources that provide more help and information (I highly recommend it for DOM).

Open a new Replit for each practice.

Practice 1

In HTML, create any element. In JavaScript find the element and print it out into the console. Change what the HTML says by using the script tag in HTML.

Practice 2

Create a button element in HTML. In JavaScript create an event listener and handler that will change what the button looks like when clicked.

Practice 3

Make a dynamic website that makes changes based on how the user interacts with it. Ex. personality quiz, web game, etc.

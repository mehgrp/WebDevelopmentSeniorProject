HTML

HTML stands for Hypertext Markup Language. Analogous to the body’s skeleton/foundation of a website. Not an actual programming language like Java, PHP, Python, etc. All websites are built with HTML.

Attributes

Attributes provide additional information or properties of elements. They are always found in the opening tag or start of an element, and the value of the attribute is put in quotes. They can be added to any HTML element. The most common attributes are seen paired with link and image tags; href and src attributes respectively and class attributes. Nesting

Nesting is when 1 or more elements live inside another element. A common and easy example of this is when a word/phrase in a paragraph is linked/bolded/italicized.

Containers

Notice how when nesting elements, closing tags are in reverse order of the opening tags. Nesting also typically involves the use of container elements. Class Attribute

There’s another way to get more specific with formatting and style. The class attribute allows you to create more specific elements, CSS and JS can call on the class name and affect only the elements with the specific class name.

Block vs. Inline Layout

Not all elements behave the same way regarding layout. Layout behavior can be broken down into block vs inline. Block elements will stack on top of each other, w/ each one starting and ending on its own line. However inline elements take up as much space as it needs, and are more commonly found nested inside a different element. Most elements are block elements; some common ones are h1-h6, p, and div. Common inline elements are a, strong, and button

Practice

The exercises below are optional, and are meant to help you practice your newly learned skills. Each option increases in difficulty. There is also a video tutorial available if you would like to see an example of how to complete each practice. There is a link at the bottom of every page to additional resources that provide more help and information.

Practice 1

Make one heading of any size (h1-h6) at the top. Then put in an image of your choosing. Add some paragraph text after that. Then make another heading of a different size than the first one.

Practice 2

If you completed the first practice then use the same Replit. If not, open a new one. Make any type of HTML list that contains at least 2 links. Then make a new paragraph element with any text but bold or link at least one word in the paragraph.

Practice 3

Make a new Replit. Try to create an About page using many different elements. This could be about a famous figure, a topic of interest, yourself, etc. It has to have at least 3 images, a few facts or a short bio, and two links or buttons (or 1 link, 1 button). Try to use the class attribute/container elements (This will be helpful when we build on it in CSS).
